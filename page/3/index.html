这里有飞鸟、游鱼，
这里有青柏、香樟，
这里丛生着杂草、杂木，
这里藏着被岁月掩埋的生活，满目苍凉，
却生意盎然。

我踏雪而来，
肃穆、愤懑又无奈。
留在这里的砖瓦将永远——
看着生命的兜兜转转。
我乘船而去，
竹篙击水，
漾开一圈圈经久不息的涟漪与愁绪。

我正慢慢地靠近你，
从未曾真正的接近你。
身着长袍的惴惴不安，
乃至深夜诉于枕边人的秘语与心酸，
都将在飞鸟的欢鸣里消散，
嘲讽的是生命将死在生命的连延。

我有一朵花

发表于 2020-11-09 分类于 生活 Valine：
本文字数： 195 阅读时长 ≈ 1 分钟

我有一朵花，
它是黄的，蓝的，紫的，妖艳的红的。
它或许只是白的，透明的。
因为它不是一朵花，因为它是一朵花。

它喜欢阳光普照，世界的通透。
它喜欢细雨乘风，偶尔的温柔。
它喜欢蝴蝶恰巧停在它的鼻头。
它喜欢蜗牛定然驻在它的心头。
它喜欢的很呢，它喜欢着吗？

稚嫩的花瓣，不提防试探。
来客大摇大摆地走了进来，
环顾空荡荡的花骨朵，
冷哼一声——
“呵，又是一个没心的家伙”
