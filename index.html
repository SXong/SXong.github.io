生命潜入蔚蓝的天空，
撕开一道平整的伤口。
什么在急切的涌动，
风在嘶鸣。

展臂、前冲、挣脱，
恰是新的轮回。
爆裂肉体出的灵魂，
在轮回中升华、死去。
或是沉沦在广阔的空间，
痴迷于宁静时间的神秘。

自由，
辽阔的自由，蔚蓝的自由。
当你随万道金光泼洒而下，
生命永不自由。

背景

管道中反射系数与传输系数的计算一直与实验有很大的偏差，但计算得到的实验测点处声压级和实验测量值契合良好。考虑采用有限元方法对相当物理问题进行计算，并比较声场，以验证计算的可靠性。如果声场具有较高的一致性，可以进一步搜索实验测试的标准文件计算系数。目前，将利用COMSOL软件求解带声衬的管道声场，声场控制方程为Helmholtz方程，声衬用阻抗边界条件代替，暂考虑流动静止的情形，利用COMSOL的压力声学模块进行求解，后续再考虑复杂流动的情形。

需要给你找到成长的空间

发表于 2021-05-09 更新于 2021-05-12 分类于 生活 Valine：
本文字数： 941 阅读时长 ≈ 2 分钟

最近难得有段时间心静了下来，看了一些书，找到了一些以前的感觉。不得不说读书还是有用的，可能最后并不能详细记住什么，但如果思考了还是有点感悟的。
感慨比较多的是《鳗鱼的旅行》，全书奇数章节讲述了鳗鱼整个人生旅行以及人类对它们习性研究的历程，偶数章节是作者帕特里克·斯文松和其父亲之间的故事。本身鳗鱼从柳叶体到玻璃鳗，再到黄色鳗和银色鳗的传奇经历就很引人入胜了，无数研究工作者前赴后继地从事鳗鱼的研究中，这是一项科学研究，也是人类对生命情感的敬畏，人类不明白是何种力量使得鳗鱼奔波千里去成长，又是何种力量使得它以生命为代价回归出生地去繁衍。关于子与父的故事也充满了温情，直接触碰心里最柔软的地方，甚至会想起很多小时候与父亲的故事，可惜我与父亲相处的经历不是特别的多，但也是现实中家庭无奈的选择。我很幸运，我的父亲虽不善言语但对于我的爱渗入到我生活中的诸多细节，巧合的是在我很小的时候，我便能感受到这种爱意，可能我的父亲、母亲和姐姐的善良与温柔或多或少地影响着我怎么去感受这个世界。我很感激，虽然有时会感觉有些沉重，但人不能贪得无厌。朱自清的父亲在临别时诸多不舍，翻过月台买来了橘子。而我的父亲在母亲各种劝阻下还是偷偷地买来了烟花，当母亲笑骂捧着烟花满脸微笑逗我开心的父亲时，五岁的我突然意识到烟花也不是那么的重要。当我上大学时，每次远行时书包和行李箱里面会被父母亲偷偷放满了硬币，被我姐塞满了新的衣物与各种各样的零食，要知道这些硬币除了解决旅途中突发的窘迫外可能并没有多大的用途，但这些东西属实支撑着我勇敢地面对困境，乃至不怯弱地去担当。每个人都有自己的故事，每个人都有自己的梦想，每个人都有自己的成长，鳗鱼的路无论是情感还是命运在操纵，它们乃至研究它们的科学家们至少找到了人生的使命，同样我们也要时刻问问自己有没有找到自己的使命，有没有给自己找到你希望的成长空间，人生漫长，从心就好。对于书中子与父在小溪旁的故事，说实话我是很羡慕的，但也只是如此而已。就像鳗鱼一样，每一条鳗鱼都有一个传奇的人生，有的甚至在深井中等数十年去寻求蜕变与归途，但最后它们一样向马尾藻海前进，我们同样的幸福。
我又要重新出发了，哈哈。还是要不停地调整自己的状态去努力的，没有全身心的投入是事实，我会努力的，哈哈。

win10采用Microsoft帐户登录无反应

如果网络没有什么问题，修改一下DNS服务器就好了。
在右击网络图标，然后选择“网络和Internet设置”-“更改适配器设置”，此时右击当前的网络连接并点击属性。
在弹出的窗口中，选中Internet协议版本4（TCP/IPV4）并点击属性，在首选DNS服务器中输入4.2.2.1，在备用DNS服务器中输入4.2.2.2，并点击确定。

突然有些迷茫

发表于 2021-02-19 分类于 生活 Valine：
本文字数： 618 阅读时长 ≈ 1 分钟

没想到我也会有今天，整天像是打了鸡血的人，哈哈~
我明白并非所有的努力都会换来你期许的结果，如果你不能好好审视自己所做的努力。如果不能像我之前说的那样将虚妄之花掰开成一片片明艳柔软的花瓣，来自虚无的恶魔就会悄悄地找上你，所谓迷茫正是如此。就像突然想起一个人一样，突然感觉所做的事情意义不大，这种感觉很是奇妙，但这奇妙之中也有些奇妙的道理。且不论这种感觉为何而来，但这个念头出现之后就是回味，回味真是个好词。我的大脑在提醒我的状况不对了，我需要一些改变。如果说迷茫是找不到追求的意义、生命中的使命，那么回答此刻出现迷茫感受的缘由或许是走出困境的关键。现在我正努力将自己从失衡状态恢复过来，我坚持跑步，时不时提醒自己读书，就是想让自己不要一直被琐事所围绕。有些事所给的回报的确很少，这是正常的。这个世界并不是所有人都能去做自己想做的事情，这是合理的，坚持有时就意味着巨大的放弃，这种程度的自由是一种理想也是一种伟大。虽然我本身有点理想主义但并非是那种极致浪漫的理想主义者，我需要完成一些事情，纵使可能会花费我很多时间。这其间这所谓突然的迷茫某种程度上算是一种“应运而生”，很幸运我明白我现在的困境。我需要做的事情是将这些事情尽快解决，然后抽出时间去做更值得去做的事情，去做更有温度的事情。
我会好好的生活，我感谢这些负面情绪（或者也不算负面）时刻在提醒着我，我不容许我这般炙热的生命丧失活力，我的生命不只要温暖，我需要热烈，坦坦荡荡，热热烈烈。

总是需要面对的

发表于 2021-02-17 更新于 2021-02-18 分类于 生活 Valine：
本文字数： 220 阅读时长 ≈ 1 分钟

生活总是离不开远行，所有的相遇与告别都是那么的奇妙。每次离开归家和离家的旅途总是有点伤感，我改变不了什么，人是要长大的啊，可何时才是真正的长大。
我又在追逐什么？我热爱着什么？我需要做些什么？我在面对什么？社会父母的期许？自己内心的渴望？我把我过得越来越虚无。我需要一步一步好好走下去，我不知道最后的最后在哪，或许什么都没有，但是我需要走下去，一步步的走下去。
人总是需要面对虚无的希望的，人总要学会把这朵虚妄之花掰开成一朵朵明艳柔软的花瓣的。
